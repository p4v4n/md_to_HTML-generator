# A Basic Markdown to HTML Generator 

Works for majority of the markdown syntax.

Doesn't support nested block quotes and direct HTML syntax yet.

## Installation

Requires **lein** and **clojure** to run.

## Usage

    lein run "test.md"
## Examples

Provided in the base directory. 
`test.md -> result.html`
